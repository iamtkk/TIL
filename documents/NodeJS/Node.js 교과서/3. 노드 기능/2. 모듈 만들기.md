- 모듈을 만들어서 관리할 수 있음

```javascript
const odd = '홀수 입니다.';
const even = '짝수 입니다.';

// module.exports = {
//     odd: odd,
//     even: even
// }
// 최신 문법에서는 객체에서 `key`와 `값 변수`가 같으면 문자를 하나로 합칠 수 있다.
module.exports = {
    odd,
    even
}

// 객체를 module.exports에 대입해준것이다. 값의 제한은 없다. 넘기고 싶은 데이터를 정의. 배열을 넘겨줘도 된다.
```
